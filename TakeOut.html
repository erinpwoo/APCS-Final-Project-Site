TakeOut Objects

A takeout object is the main object that inspired this entire project. It extends the abstract RestaurantOrder class and represents a single takeout item and carries attributes such as size, type of order, price, etc. It also contains methods that either retrieve data about a takeout object or modifies that data. It is later utilized by the OrderDoordash class, which initializes multiple TakeOut objects.
